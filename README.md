# 📋 HabitCore API

## 📖 Sobre

Este projeto consiste em uma aplicação web simples desenvolvida em **Node.js** com o objetivo de praticar conceitos fundamentais de desenvolvimento **backend**.

A aplicação implementa um sistema básico de **autenticação de usuários e gerenciamento de tarefas**, permitindo que cada usuário tenha seu próprio dashboard com suas tarefas.

Funcionalidades implementadas:

- Registro de usuários
- Login de usuários
- Criação de tarefas
- Listagem de tarefas por usuário
- Exclusão de tarefas
- Sistema de sessão simples utilizando LocalStorage

Durante o desenvolvimento foram praticados conceitos como:

- Estruturação de rotas com Express
- Comunicação entre **frontend e backend**
- Manipulação de requisições HTTP
- Implementação de **CRUD**
- Organização básica de um servidor Node.js
- Uso de JavaScript assíncrono (async/await)

---

## 🛠 Tecnologias utilizadas
